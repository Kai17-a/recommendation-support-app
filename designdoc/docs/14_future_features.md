# 将来機能

以下は初期リリースの必須範囲ではない。

## 1. 高度な検索

- スキル根拠付き検索
- 案件経験検索
- 部署内候補者検索

ただしランキングや自動推薦は行わない。

## 2. レビュー協働

- 複数上司によるコメント
- 承認・差し戻し
- 人事レビュー
- 提出版管理

## 3. 出力

- Markdown出力
- DOCX出力
- PDF出力
- 会社指定様式への変換

## 4. 外部システム連携

- 人事システム
- 案件管理システム
- スキル管理システム
- 社内認証基盤

## 5. メンバー本人利用

将来的に本人入力をシステム内へ移行する可能性はあるが、権限と承認フローを別途設計する。

## 6. 案件タイムライン

案件内の成長や役割変化を視覚化する。  
必須機能ではない。

## 7. エビデンス拡張

初期はMarkdownと登録データを根拠とする。  
将来的に会社方針が定まれば他形式を検討する。
