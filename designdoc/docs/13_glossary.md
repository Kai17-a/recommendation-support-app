# 用語集

| 用語               | 定義                                                       |
| ------------------ | ---------------------------------------------------------- |
| メンバー           | 推薦対象となる社員                                         |
| 部長・マネージャー | メンバー情報と推薦業務を担当する利用者                     |
| システム運用者     | AI設定、ユーザー管理、監査、物理削除を担当する利用者       |
| 案件経験           | 特定メンバーが特定案件で得た経験のまとまり                 |
| 案件報告           | 定期、終了、任意の単位で登録される活動記録                 |
| 人物評価           | 上司が時系列で記録する人物面の評価                         |
| 推薦プロジェクト   | 推薦文の作成、修正、確定を管理する単位                     |
| 根拠               | AI生成文やスキル候補を裏付ける案件報告、評価、コメント     |
| AI分析             | 案件情報から事実、スキル候補、強み候補、根拠を整理する処理 |
| AI Gateway         | OpenAI互換APIを提供し、複数LLMへの接続を仲介する基盤       |
| Custom Provider    | OpenAI互換仕様に沿ってAI Gatewayへ接続するプロバイダー設定 |
| 論理削除           | データをDBから消さず、削除済みとして非表示にする方式       |
| 物理削除           | データをDBまたは保存領域から実際に消去する処理             |
| 監査ログ           | 誰がいつ何を変更したかを記録するログ                       |
