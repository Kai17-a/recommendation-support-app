# システム概要

## 1. システム名

推薦業務支援システム

## 2. システムコンセプト

> 推薦業務という業務プロセスをデジタル化し、その一部をAIで支援する。

## 3. 背景

部長・マネージャーが、メンバーを役職、社内案件、顧客案件などへ推薦する際、案件経験や人物評価を参照しながら毎回ゼロから資料を作成している。

この作業には以下の課題がある。

- 過去案件や実績を探す負荷が高い
- 推薦目的ごとに情報を取捨選択する必要がある
- 推薦文の構成や表現に時間がかかる
- 根拠となる情報が複数箇所に分散する
- 担当上司の記憶や経験に依存する

## 4. 目的

本システムの目的は、推薦業務を完全自動化することではない。

以下を支援し、上司の作業負荷を軽減する。

- メンバーの案件経験を時系列で管理する
- 人物評価を定期的に蓄積する
- スキルと根拠を紐付ける
- 推薦目的に応じて必要な情報を抽出する
- 根拠付きの推薦文ドラフトを生成する
- 上司がドラフトを修正・確定する

## 5. 対象利用者

### 部長・マネージャー

- 直属メンバーおよび自部署メンバーの管理
- 案件経験・報告・評価の登録
- Markdown報告の取り込み
- AI分析の任意実行
- 推薦プロジェクトの作成
- 推薦文の生成・修正・確定

### システム運用者

- ユーザー・部署管理
- AI Gateway接続設定
- AIモデル設定
- Markdownテンプレート管理
- データ保持期間管理
- 監査ログ確認
- 物理削除の実行

### メンバー本人

システムを直接利用しない。  
指定されたMarkdownテンプレートへ案件報告を記載し、部長・マネージャーへ提出する。

## 6. 対象外

初期バージョンでは以下を対象外とする。

- メンバー本人によるログイン・直接入力
- AIによる推薦可否の判断
- 人材ランキング
- 適性スコア
- 推薦文品質スコア
- 複数メンバーの自動比較
- Word、Excel、PDF、画像の取り込み
- 案件共通マスタによる全社案件管理
