# システム概要

## システムコンセプト

> 推薦業務という業務プロセスをデジタル化し、その一部をAIで支援する。

## 利用環境

- 閉じた社内ネットワークで利用する
- インターネットへ公開しない
- 不特定多数の利用者を想定しない
- 業務適合性とAI機能を検証するプロトタイプとする
- セキュリティ要件は必要最低限とする

本番展開や外部公開へ進む場合は、認証、認可、秘密管理、監査、暗号化を再設計する。

## 初期版の対象外

- インターネット公開
- 外部利用者向け認証
- WAFや公開API Gateway
- Secret ManagerやVaultの必須化
- 高可用性構成
- AIによる推薦可否判定やスコアリング
